# 2025年年终总结

我来跟风写一下我的2025年年终总结。

今年开始我慢慢开始撰写我自己的日记。买了不少笔和本子。但是日记写的却是断断续续的。特别是海南大气实验的这一个月，感觉根本就没写几次。
那段时间过的也是非常不规律。每天早起待命，等待天气，地基雷达，空域申请的通知，面对飞行任务能否开展的不确定性，特别是几次在前往机场的路上，甚至已经上飞机准备了，
收到了空域限制飞行取消的通知，感觉心力憔悴，自己整个人也变得迟钝起来了。大片的时间在干干等待，能够组合成不小规模的学习和工作时间也被自己放手流逝。
