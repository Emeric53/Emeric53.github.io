# 2026 年的春天，我不知道会是这样
一转眼，2026年马上要过去三分之一了，感觉事情发生了很多，自己也有了很多的变化和感触。无论如何，很久没有撰写新的博客了，难得收拾一下心情，重新记录一下。

## 新的开始
从春节前夕接到第二次面试邀请，到过一段时间拿到 PhD offer，到过年期间准备签证材料，再到回北京进行签证预约以及真正拿到签证，现在回想这一段时间，还是一些虚虚实实的。特别是到了面签的那一天，我也不知道为什么会如此紧张。从另一个角度来说，我也感觉一下子生活要有了如此大的变化，自己也像是被推着往前走。但是我希望能把这个转变作为个人生命旅程的一次重要的转折，还记得自己总是不满意自己的生活方式，看的'How to change' 这本书里面写的，希望去美国读书能作为一个新的 blank slate，让我的生活节奏，处事方式，社会化程度都能有一个比较大的改变。其实我已经特别想去了，但是又挺害怕的，感觉到时候会面临各种各样的问题和挑战。新鲜的事物，新鲜的生活，新鲜的节奏和文化，我能适应嘛？

## 玩不转的人际关系
我觉得我就是处不了很多朋友，这是我这段时间的感悟。我没办法对大家差不多的态度，因为交际圈中我的喜欢程度总是层次分明。Social Wealth 中的三大支柱：Depth，Width，以及 Earned Status 我感觉我对 Depth 的重视程度是最高的，所以一般也只会想把我的时间花给那些在这个圈层里面的人。这段时间的交往经历也让我学习到了一些东西，长了一些教训。我也更清楚知道和感受到，有些事情你做了，不一定会收获什么，但是大概率也不会损失什么，但是你不做，你一定收获不到任何东西。面对自己，要尽量进行个人行为的墒减，而面对社会，把自己放到一个墒增的环境和情景下，才能锻炼到墒减的能力。对于以后的个人行为复盘，我会遵循这个原则去做的。

## 生活和工作
马上快到我的生日了，我也切切实实进入了二十代的后半段了，在目前这个过渡和转折的阶段，我觉得我做错了很多事情，也浪费了不少时间。转头来看，其实这就是一段给我试错和调整的时间。最近总是对自己的身体状态比较关注，可能是因为确实不舒服的情况拖了太久，此外，睡眠问题引发的皮肤状态，心情和精力活跃程度也总是在反反复复想要修正，却又一遍一遍复发。减少身体的疼痛以及自在性应当是首要目标。工作时间的专注度和自主度是我考量的第二件事情，其实我真的很想断舍离，莫名其妙的就会不断购买新的电子设备，新的产品，新的装饰品，确实有点消费过度了。自己总是给自己创造 Distraction，想要有所产出却一直停滞不前的状态也总是得不到改变。博客更新这件事情不做，就会一直不想做，但是做了其实也没什么负担，心里反而会舒服一些。

## Summer wants some more
还记得我的 AirPods Pro 1st generation 上的镌刻，就写了 [summer wants some more] 这句话。我其实很喜欢这句话，我也很喜欢夏天（除了越来越热）。每每看到想到这句话，总会把我牵引回2019年，当时的我感觉没有烦恼没有忧虑，至少状态是好的，心也是有渴望的。希望接下来的夏天，我也能慢慢转变，我真的是一个很贪心的人，但是只停留在内心的贪心，而不是行动上的贪心，未免总会重蹈覆辙。期盼这个夏天的到来。
